defaultCacheDirectory

var defaultCacheDirectory: File

The default File directory where to store cache. Note: changing default directory may require permission.READ_EXTERNAL_STORAGE and permission.WRITE_EXTERNAL_STORAGE

Default is initDefaultCacheDir

When this value is changed, cache will be removed from the old directory. If new directory not File.exists, it will be created.

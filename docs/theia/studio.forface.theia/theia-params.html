theiaParams

fun theiaParams(block: ExtraParams): ExtraParams

A function that takes an ExtraParams block and return the same block. It's purpose is to create a block that will create TheiaParams that can be applied on another instance of AbsTheiaBuilder.

Example:

val extraParams = theiaParams { imageUrl = someImageUrl shape = Round } someImageView.theia( extraParams ) { scaleType = Center }
